# Contributing

## Dependencies

### Run/build/package dependencies

- `node` (see `package.json` for current version)
- `npm`

### Test dependencies (Linux-only)

#### Debian/Ubuntu

- `xvfb` - required for `npm test`

#### Fedora

- `xorg-x11-server-Xvfb` - required for `npm test`

#### Arch

- `xorg-server-xvfb` - required for `npm test`
